# To-Do List App 📋
A command line To-Do List application built with Python.

## Features
- Add new tasks
- View all tasks with status
- Mark tasks as complete
- Delete tasks
- Error handling for invalid inputs

## How to run
python todo.py

## Tech used
- Python 3
- Lists and Dictionaries
- Functions and Error Handling
